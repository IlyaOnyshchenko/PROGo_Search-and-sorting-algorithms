Ближайшее число
Напишите программу, которая находит в последовательности элемент, самый близкий по величине к данному числу.
Входные данные
В первой строке задается одно натуральное число N, не превосходящее 1000.
Во второй строке содержатся N целых чисел, каждое из которых по модулю не превосходит 1000.
В третьей строке вводится одно целое число x, по модулю не превосходящее 1000.
Выходные данные
Вывести значение самого ближайшего элемента к x. Если таких чисел несколько, вывести наибольшее из них.

Sample Input 1:

3
2 10 5
6

Sample Output 1:

5

Sample Input 2:

3
2 10 5
1

Sample Output 2:

2

Sample Input 3:

3
2 10 5
8

Sample Output 3:

10

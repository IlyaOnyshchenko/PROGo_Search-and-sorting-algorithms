Поиск-2 
Напишите программу, которая определяет, встречается ли заданное число x в последовательности чисел.
Входные данные
В первой строке задается одно натуральное число N, не превосходящее 1000.
Во второй строке вводятся N целых чисел, каждое из которых по модулю не превосходит 1000.
В третьей строке содержится одно целое число xx, не превосходящее по модулю 1000.
Выходные данные
Вывести YES , если число xx встречается в данном массиве, и NO в противном случае.

Sample Input 1:

6
1 2 3 4 5 6
5

Sample Output 1:

YES

Sample Input 2:

3
2 3 10
5

Sample Output 2:

NO

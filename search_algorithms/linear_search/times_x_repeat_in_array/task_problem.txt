Поиск
Напишите программу, которая определяет, сколько раз встречается заданное число x в данном массиве.
Входные данные
В первой строке задается одно натуральное число N, не превосходящее 1000.
Во второй строке вводятся N целых чисел, каждое из которых по модулю не превосходит 1000.
В третьей строке содержится одно целое число x, не превосходящее по модулю 1000.
Выходные данные
Вывести одно число – количество чисел, равных x.

Sample Input 1:

6
6 8 5 1 5 4
5

Sample Output 1:

2

Sample Input 2:

3
1 2 3
5

Sample Output 2:

0

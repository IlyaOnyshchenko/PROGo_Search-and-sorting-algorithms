На вход программы поступает последовательность из N целых положительных чисел. Все числа не превышают 1000.Выведите количество чисел, которые встречаются ровно 2 раза.
 
Входные данные
В первой строке задается одно натуральное число N, не превосходящее 100000.
В каждой из последующих N строк записано одно целое положительное число, не превышающее 1000. 
 
Выходные данные
В качестве результата программа должна вывести количество чисел, которые встречаются ровно 2 раза.

Sample Input 1:

5
1
2
3
2
1

Sample Output 1:

2

Sample Input 2:

1
1000

Sample Output 2:

0

Sample Input 3:

1
1
Sample Output 3:

0

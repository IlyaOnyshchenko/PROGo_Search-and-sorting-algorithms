Числа анаграммы
Число называется анаграммой другого числа, если оно может быть получено любой перестановкой его цифр.
Входные данные
Даны два натуральных числа на отдельных строках. Каждое из чисел не превышает 10^9.

Выходные данные
Требуется вывести "YES"  – если введенные числа являются анаграммами друг друга, "NO"  – если нет.

Sample Input 1:

524
452

Sample Output 1:

YES

Sample Input 2:

554
452

Sample Output 2:

NO

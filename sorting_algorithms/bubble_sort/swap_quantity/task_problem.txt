Количество обменов
Дан массив целых чисел. Определите, сколько обменов сделает алгоритм пузырьковой сортировки по возрастанию для данного массива.
Входные данные
На первой строке дано число N (1≤ N ≤1000) – количество элементов в массиве.
Во второй строке вводятся N целых чисел, каждое из которых по модулю не превосходит 10^9. Гарантируется, что все элементы массива различны.
Выходные данные
Выведите одно число – количество обменов пузырьковой сортировки.

Sample Input 1:

5
15 1 5 568 13

Sample Output 1:

4

Sample Input 2:

3
1 2 3

Sample Output 2:

0

Sample Input 3:

5
1 2 3 4 5

Sample Output 3:

0
